# VTS_XYPlugin
用于Bilibili的Vtuber Studio的插件

## 下载
[releases][1]

## 插件说明
获取B站直播数据，并根据付费礼物播放对应动作等，可以绑定多种动作，可以显示多条日志。

## 功能建议
可以直接提Issues或者在B站私信我，我有空的时候会研究。

## 修改说明
由于使用了资源商店的插件，所以git上的源文件并不是完整的文件。
如果你需要修改并编译，需要在package manager中导入odin和easy save 3两个插件。
